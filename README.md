# 🩺 Diabetes Prediction

## 📌 Overview
This project is a Machine Learning-based system that predicts whether a person has diabetes based on medical attributes.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Features Used
- Glucose Level
- Blood Pressure
- BMI (Body Mass Index)
- Age
- Insulin Level
- Skin Thickness

## 📊 How it Works
1. Data preprocessing (handling missing values)
2. Feature selection
3. Model training using classification algorithms
4. Prediction of diabetic or non-diabetic condition

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Enter input values to test prediction

- Not Diabetic ✅
